## 📥 How to run

```bash
# Clone the project:
$ git clone https://github.com/rebecagrn/fakeflix.git

# Install the dependencies
$ yarn

# Run in browser
$ yarn start

```

# :rocket: Features

- Search movies
- Favorite movies
- Move to Watch later
